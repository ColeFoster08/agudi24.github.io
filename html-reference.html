HTML Element

Description

The HTML element is used to tell the browser that this is an HTML document.

Syntax

<html>
*code*
</html>

Example of usage

This element is used in all html documents and is always at the beginning and end of the document, but it does not appear on the actual document.

Rendering

It doesn't show on the actual document.

Header Element

Description

The <header> element represents a container for introductory content or a set of navigational links.

Syntax

<head>
*code*
</head>

Example of usage

This element is used in all html documents. It is used mostly to write style code for the page, such as the color of the background.

Rendering

It doesn't appear on the document.

Title Element

Description

The Title element is used to name the document. It appears on the tab of the top of the browser.

Syntax

<title>
*code*
</title>

Example of usage

This element is used in all html documents and is always near the beginning of the document. You can find the title at the tab of the document.

Rendering

It shows "Your Title" at the tab of the document.

Body Element

Description

The Body element is used to identify the text that is going to be written on the document.

Syntax

<body>
*code*
</body>

Example of usage

This element is used in all html documents and is always near the beginning of the document. You can find this element at the beginning of the code and at the end of it.

Rendering

It doesn't show on the actual document.

Headings Element

Description

The headings element is used to give the heading on the document. You can change the size of your heading by changing the number (h2, h3, h4, h5, h6).

Syntax

<h1>
*code*
</h1>

Example of usage

This element is used in almost all html documents and is mostly near the beginning of the document. When you have a big idea on the topic you are writing about, you wuld most likely write it on the document using this element.

Rendering

*Code*

Paragraph Element

Description

The paragraph element is used to write things on the document. It is usually used when you are writing a lot about a certain topic.

Syntax

<p>
*code*
</p>

Example of usage

This element is used in all html documents and is usually after a heading in the document. When you are writing a paragraph, you would most likely use this element.

Rendering

I just wrote this using the paragraph element.

Break Element

Description

The HTML <br> element produces a line break in text. It is useful for writing a poem or an address, where the division of lines is significant.

Syntax

*code*
<br>
*more code*

Example of usage

This element is used in almost all html documents and is usually after a paragraph or line of writing in the document. When you are writing an address or a poem, you would most likely use this element.

Rendering

Hi.
Hi Again.

Line Element

Description

The <hr> (line) tag defines a thematic break in an HTML page (e.g. a shift of topic).The <hr> element is used to separate content (or define a change) in an HTML page.

Syntax

*code*
<hr>
*more code*

Example of usage

This element is used in a lot of html documents and is usually after the end of a paragraph or line of writing in the document. When you are writing an explanatory article, you would most likely use this element.

Rendering

*

*

Comment Element

Description

The comment tag is used to insert comments in the source code. Comments are not displayed in the browsers. You can use comments to explain your code, which can help you when you edit the source code at a later date. This is especially useful if you have a lot of code.

Syntax

<!--This is a comment. Comments are not displayed in the browser.-->

Example of usage

This element is used in lots of html documents and is usually before a paragraph or line of writing in the document. When you are writing a lot of code, you would most likely use this element.

Rendering

Comments do not show on browsers. It is located in the code.

Link Element

Description

The <link> tag defines a link between a document and an external resource. It is also used to link to external style sheets.

Syntax

<a href="url">Link Text</a>

Example of usage

This element is used in a lot of html documents and is usually before or after some text in the document. When you are writing an argumentative essay and you need some evidence, you would most likely use this element to have the evidence in the document.

Rendering

This is a link

Image Element

Description

The <img> tag defines an image in an HTML page. It has two required attributes: src and alt.

Syntax

<img src="url" height=any number width=any number>

Example of usage

This element is used in a lot of html documents and is usually before or after some text in the document. When you are writing about a specific topic, you would most likely use this element to show an image of that topic.

Rendering

List Element

Description

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag. An ordered list starts with the <ol> tag. Each list item starts with the <li> tag.

Syntax

For <ul>:
<ul>
<li>code</li>
<li>code</li>
<li>code</li>
</ul>

For <ol>:
<ol>
<li>code</li>
<li>code</li>
<li>code</li>
</ol>

Example of usage

This element is used in a lot of html documents and is usually used before or after some writing. When you are writing a recipe and you need to list out the ingredients, you would use this element.

Rendering

Unordered list:

  • Coffee
  • Tea
  • Milk

Ordered list:

  1. Coffee
  2. Tea
  3. Milk

Table Element

Description
